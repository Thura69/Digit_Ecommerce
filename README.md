
# Digit Ecommerce UI

Digit E-commerce UI with blog section, products filters and compare.


## Features

- Login
- Register
- Home
- Blog 
- Compare
- Wishlist
- Category


## Tech Stack

React.js, Tailwind, React Router Dom, Git & GitHub
